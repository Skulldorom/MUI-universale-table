# MUI Universal Table

MUI Universal Table is a reusable React table component built on Material UI.

It provides common data table features out of the box:

- Built-in search across configured columns
- Sorting and pagination
- Row selection
- Expandable sub-tables
- Refresh and lazy-loading states
- Responsive, customizable UI

Use this site to quickly get started and understand the available API.
